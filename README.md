# Custom Linux-Shortcut-Creation


1) Open up the file ".bashrc " by using the following command. 
 
	gedit ~/.bashrc ( I am using gedit , but you can use the text editor of your choice)

2) Add " alias='<Your Command> " to the file as shown in the image . 

3) To save your changes , simply source the file , i.e " source ~/.bashrc" or just open up a new terminasl and it will be automatically sourced.
